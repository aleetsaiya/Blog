## 添加 git bash
1. 點 window terminal 分頁視窗旁的 向下按鈕，選擇 `設定`
2. 新增設定檔 :
```
名稱: Git bash
命令列: C:\Program Files\Git\bin\bash.exe
圖示: C:\Program Files\Git\mingw64\share\git\git-for-windows.ico
```

儲存後就可以用 window terminal 的分頁增加一個 Git bash 了。
